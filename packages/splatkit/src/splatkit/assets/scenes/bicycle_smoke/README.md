This bundled smoke scene is a tiny COLMAP subset derived from the local
`bicycle` Mip-NeRF 360 scene used during development.

Contents:
- 6 downscaled JPEG views at 640px width
- a pruned text-format COLMAP sparse model

Purpose:
- smoke-test COLMAP loading
- notebook defaults without personal absolute paths
- small relative render and dataloader benchmarks

This asset is intentionally not a representative training or quality benchmark.
